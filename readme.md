# 📝 React Smart Todo

A complete Todo Application built using React Hooks.

This project demonstrates core React concepts like:
- useState
- Objects & State
- Arrays & State
- List Rendering
- Unique Keys
- Updating Items
- Deleting Items
- Conditional Styling

---

## 🚀 Features

✅ Add new tasks  
✅ Delete tasks  
✅ Mark task as done  
✅ Mark all tasks as done  
✅ Unique ID for each task  
✅ Dynamic UI updates  

---

## 🛠 Tech Stack

- React
- JavaScript (ES6)
- HTML
- CSS (Inline Styling)

---

## 📚 Concepts Practiced

- Functional Components
- State Management using useState
- Immutable Updates
- Array Methods (map, filter)
- Conditional Rendering
- Event Handling
